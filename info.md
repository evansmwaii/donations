# Microcontroller-based Egg Incubator

## Project Overview
### Introduction
This project introduces an automated solution utilizing a microcontroller-based system to regulate the internal environment of the incubator. By integrating sensors and actuators, the system will maintain optimal conditions without constant human supervision. It addresses the gap between manual labor-intensive methods and expensive industrial hardware by providing a cost-effective, high-efficiency alternative.

---

## Justification of the Study
This project is vital for enhancing food security and supporting the "Big Four" agenda in the agricultural sector. By automating the incubation process, farmers can achieve higher yields with minimal labor costs. This technology also benefits the engineering industry by demonstrating a practical application of embedded systems and IoT in solving real-world biological and agricultural problems.

## Technical Specifications
### Scope of the Project
1.  Construction of a thermally insulated incubation chamber equipped with heating elements and cooling fans.
2.  Implementation of a microcontroller-based circuit to process sensor data and automate environment regulation.
3.  Integration of high-precision sensors for real-time monitoring of internal temperature and humidity levels.
4.  Design of an automated motorized tray mechanism for scheduled egg turning.
5.  Development of a reliable power supply unit with surge protection for electronic components.
6.  Installation of an LCD screen and indicators for local status monitoring and system alerts.

### Required Components
#### Hardware:
* **Microcontroller:** Arduino Uno / ESP32
* **Sensors:** DHT22 (Temperature and Humidity)
* **Heating:** 60W Incandescent Bulb
* **Cooling:** 12V DC Brushless Fan
* **Turning Mechanism:** MG996R Servo Motor
* **Power Supply:** 12V, 5A AC-DC Power Adapter
* **Display:** I2C 16x2 LCD Module
* **Enclosure:** Plywood Box

#### Software:
* **Environment:** Arduino IDE / PlatformIO
* **Languages:** C / C++ Programming
* **Design:** SolidWorks / AutoCAD

## Objectives
* To implement an automated egg-turning mechanism using a servo or stepper motor to prevent embryo sticking.
* To develop a control algorithm for heat regulation using a PID or ON/OFF Controller.
* To test and evaluate the system's accuracy in maintaining the required hatching parameters.
